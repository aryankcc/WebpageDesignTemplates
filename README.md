## Summary

The Vizient Webpage Design Configurator is an internal web application that enables users to visually configure new webpage requests before submitting them through Workfront.

Instead of describing page layouts, branding, and content structure through text alone, users can interactively explore approved Vizient webpage templates, customize predefined design options, and generate a visual mockup to accompany their request.

The exported mockup serves as a visual reference for the Marketing Operations team and Advansiv during webpage creation, helping reduce ambiguity and streamline the request process.

## Problem

Creating new webpages currently relies on written descriptions submitted through Workfront. This often makes it difficult for requesters to communicate:

- Desired page layout
- Hero design preferences
- Feature section selection
- Brand color usage
- Overall visual direction

Without a shared visual reference, requests can require additional clarification and multiple revision cycles before development begins.

---

## Solution

This configurator provides a guided interface where users can:

- Browse approved Vizient webpage templates
- Preview layout variations in real time
- Explore brand-approved color palettes
- Configure hero sections and page components
- Customize typography and content sections
- Export a visual mockup (PDF, JPG, or screenshot)
- Attach the generated mockup to their Workfront request

The application is intended to improve communication—not replace the final design and implementation process performed by Advansiv.
